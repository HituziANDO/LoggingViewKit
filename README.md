# LoggingViewKit

***LoggingViewKitはユーザがどのViewをタップしたかを記録するためのライブラリです***

## Include in your iOS app

1. LoggingViewKit.frameworkをXcodeプロジェクトにドラッグ＆ドロップします
1. Build Settings > Linking > Other Linker Flagsに `-all_load` を追加します

## Supported View

- Button
- Label
- SegmentedControl
- Slider
- Stepper
- Switch
- View
